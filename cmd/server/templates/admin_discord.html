시크릿 값은 화면에 다시 표시하지 않으며 서버의 config.conf에 권한 0600으로 저장합니다.

Developer Portal의 OAuth2 Redirects에도 완전히 같은 주소를 등록해야 합니다. 경로는 반드시 /ra/discord/callback이어야 합니다.

인증 정책

  • 일반 Discord 로그인에서는 계정을 자동 등록하지 않습니다.
  • OAuth 요청은 10분 동안만 유효한 일회성 state 값으로 검증합니다.
  • Discord 액세스 토큰과 리프레시 토큰은 저장하지 않고 사용자 ID 확인에만 사용합니다.
  • Discord 로그인을 해제하거나 다른 Client ID로 변경하면 다시 연동하기 전까지 로그인이 차단됩니다.
